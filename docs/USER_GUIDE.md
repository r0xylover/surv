# Локальный запуск

1. Установите Node.js версии 20 или выше.
2. Откройте папку проекта в терминале.
3. Выполните команду `npm install`.
4. Выполните команду `npm run dev`.
5. Откройте адрес, который покажет Vite, обычно `http://localhost:5173`.

# Production-сборка

1. Выполните `npm install`.
2. Выполните `npm run build`.
3. Для проверки выполните `npm run preview`.

# Публикация на GitHub Pages

1. Создайте пустой репозиторий на GitHub.
2. Загрузите в него содержимое этой папки.
3. Убедитесь, что основная ветка называется `main`.
4. Откройте раздел `Settings` → `Pages`.
5. Укажите источник публикации `GitHub Actions`.
6. После push GitHub сам опубликует приложение.

# Резервное копирование

1. Нажмите `JSON` для резервной копии данных.
2. Нажмите `ZIP` для архива проекта.
3. Храните резервные копии отдельно от устройства обследования.

# Импорт проекта

1. Откройте приложение.
2. Нажмите `Импорт`.
3. Выберите ранее сохраненный JSON-файл.
4. Дождитесь загрузки данных.

# Установка PWA

## Android
- открыть сайт в Chrome;
- нажать `Установить приложение`.

## iPhone
- открыть сайт в Safari;
- нажать `Поделиться` → `На экран Домой`;
- запускать приложение с иконки на домашнем экране;
- для первого офлайн-использования один раз полностью открыть приложение при наличии интернета.

## Windows и macOS
- открыть сайт в Chrome или Edge;
- нажать кнопку установки в адресной строке.

# Обновление

1. Загрузите новую версию в GitHub.
2. Дождитесь завершения GitHub Actions.
3. Перезапустите приложение на устройстве.
